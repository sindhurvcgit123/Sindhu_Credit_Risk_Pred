# Credit_Risk_Project — GBM + SHAP interpretability

## Project summary
This repository contains a reproducible notebook and required deliverables for the Cultus "Interpretable AI: SHAP Analysis of a Complex Gradient Boosting Model for Credit Risk" project. Uses the `credit-g` dataset (available through sklearn).

## Contents
- `credit_risk_notebook.ipynb` — Complete notebook with data loading, preprocessing, model training, SHAP global and local analyses, and bias checks.
- `model_performance_report.txt` — Performance report (to fill with actual results after running notebook).
- `bias_analysis_report.txt` — Fairness analysis (to fill with evaluated numbers).
- `executive_summary.txt` — 500-word summary for compliance.
- `README.md` — This file.

## Requirements
Create a Python environment with the following packages:
python >= 3.8
pandas
numpy
scikit-learn
lightgbm
shap
matplotlib
seaborn
